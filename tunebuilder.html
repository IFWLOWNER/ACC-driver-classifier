ACC Tune Image Builder

Drop ACC setup JSON files, edit anything, then copy/export one clean Discord-ready image.
Note: the blue section tabs inside the tune sheet are now clickable. Download tune PNG exports the tune sheet only. Download long tune PNG exports the full long tune sheet only, using an auto-tight crop so there is no left-side controls and no wasted right-side space.

1. Add files

Drop setup JSON files or a track folder here
You can add multiple tunes, or drop/select a folder named after the track, such as brands_hatch, zolder or red_bull_ring.
Folder upload: if you upload a folder named after an ACC track, every JSON inside will be loaded with that track selected automatically. Tick the tunes you want, then use Save selected to database.
Console screenshots cannot be read perfectly in a plain offline HTML page. Use Blank console tune, then type/copy the values from the image into the editable cards. The main export is the tune sheet on the right. Exports render as long PNG files for Discord posting, and the long tune export excludes the left-side controls and uses an auto-tight crop to the tune sheet width.
